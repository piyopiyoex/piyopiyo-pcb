# KiCad プロジェクト

このディレクトリには、Piyopiyo PCB の KiCad プロジェクトファイルをまとめています。

主なファイル:

- `.kicad_pro` — プロジェクト本体
- `.kicad_sch` — 回路図
- `.kicad_pcb` — 基板レイアウト
- `Library.pretty/` — フットプリントライブラリ
- `fp-lib-table`, `fp-info-cache` など — KiCad が生成する設定・キャッシュ

KiCad でこの版の回路やパターンを編集したい場合は、このディレクトリ内の `.kicad_pro` を開いてください。
