# ハードウェア変更履歴

このファイルは、各バージョン間の差分を簡潔にまとめたものです。
詳細は各バージョン配下の KiCad データおよび製造データを参照してください。

## v1.9

- 回路の変更はありません
- U2 の EPAD（放熱パッド）のパターンを修正し、放熱効果を向上
  - 実際にはほとんど発熱しないため、動作上の影響は小さいと見込まれます

## v1.8

- 令和八年一月 ダビデさん来日記念リリース

## v1.7

- J12 にてシリアル通信の TX/RX が利用できるように修正
- Serial1 の TX/RX コネクタに関し、ループバックテストで動作確認済み
  - T_CS と RX が重複しているため、3.5 インチ LCD 使用時は受信できません
  - 8PIN 接続の LCD では送受信の両方が利用可能です
