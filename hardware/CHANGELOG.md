# ハードウェア変更履歴

このファイルは、各バージョン間の差分を簡潔にまとめたものです。
詳細は各バージョン配下の KiCad データおよび製造データを参照してください。

## v1.9

- 回路の変更はありません
- U2 の EPAD（放熱パッド）について、放熱効果を高めるためのパターン修正を行いました
  - 実際にはほとんど発熱しないため、動作上の影響は小さい想定です
- v1.9 を実装して動作確認を実施しました
- v1.7 で追加した Serial1 の TX/RX コネクタについて、ループバックテストで動作確認しました
  - T_CS と RX が重なっているため、3.5 インチ LCD 利用時は受信できません
  - 8PIN 接続の LCD では送受信が利用できます
