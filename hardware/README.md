# hardware

`hardware/` には、Piyopiyo PCB のハードウェア設計データをまとめています。

主な内容は次のとおりです。

- `piyopiyo-pcb.vX.Y.pdf`
  回路図・基板レイアウトの PDF

- `piyopiyo-pcb.vX.Y.bom.xlsx`
  部品表（人間が読む用）

- `piyopiyo-pcb.vX.Y.bom_jlcpcb.csv` / `piyopiyo-pcb.vX.Y.cpl_jlcpcb.csv`
  JLCPCB 向けの BOM / CPL

- `piyopiyo-pcb.vX.Y.gerber.zip`
  基板製造用 Gerber 一式

- `kicad/`
  KiCad プロジェクト一式

- `reference/`
  搭載モジュールなどの参考資料
